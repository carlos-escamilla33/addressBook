
# addressBook 📖

This app simply manages the address of employees in an organization.


## Tech Used

**Client:** Backbone.js, Underscore.js, jQuery


## 🚀 Run Locally

Clone the project

```bash
  git clone https://github.com/carlos-escamilla33/addressBook
```

Go to the project directory

```bash
  cd addressBook
```

**Go into index.html**
